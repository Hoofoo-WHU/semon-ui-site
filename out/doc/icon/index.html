Icon 图标

常用的图标集合。

代码演示

基本用法

通过type属性指定图标名称, spin属性使图标旋转。

import { Icon } from '@semon/semon-ui'

ReactDOM.render(
  <div>
    <Icon type='user' />
    <Icon type='search' />
    <Icon type='code' />
    <Icon type='check-circle-fill' spin />
  </div>,
  container
)

API

属性 说明 类型 可选值 默认值
type 类型 string types提示的所有值 -
spin 旋转 boolean true false false
